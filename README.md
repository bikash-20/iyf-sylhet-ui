# IYF Sylhet UI

A modern landing page for the ISKCON Youth Forum of Sylhet, built with clean HTML, CSS, and subtle motion enhancement.

## Features

- Full-bleed deity hero section with glassmorphism overlay
- Compact responsive navigation with pill-shaped buttons
- Unified page flow using a continuous frosted glass shell
- Interactive cards, timeline, and media section
- Smooth reveal animations via JavaScript

## Notes

This is a static site designed for visual presentation and responsive browsing. The UI emphasizes devotional warmth with soft gradients, elegant typography, and polished glass effects.
